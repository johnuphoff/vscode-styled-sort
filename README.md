# vscode-styled-sort
Opinionated rule sorting for styled-components.

![animated.gif](https://s33.postimg.cc/c1k5urvm7/animated.gif)

## Install
Inside VSCode, press Ctrl+P, and enter:
```
ext install vscode-styled-sort
```
It should be the top result.
